## 系统概述

这是一个为 Godot 4.x 设计的**高度抽象、可复用的任务系统**,适用于各种类型的游戏项目。

### 核心特性

✅ **高度抽象** - 核心逻辑与具体游戏解耦  
✅ **高度灵活** - 组件化设计,易于扩展  
✅ **可复用性** - 可直接用于多个项目  
✅ **数据驱动** - 通过配置定义任务  
✅ **事件驱动** - 松耦合的事件系统  
✅ **完整功能** - 支持目标、条件、奖励、任务链等

### 支持的任务类型

- 主线任务
- 支线任务
- 日常/周常任务
- 成就系统
- 赏金任务
- 任务链(顺序/分支)

## 核心架构

### 系统层次结构

```
任务系统
├── 数据层 (TaskData)          - 任务配置数据
├── 逻辑层 (TaskInstance)      - 运行时实例
├── 管理层 (TaskManager)       - 全局管理器
└── 事件层 (QuestEventBus)     - 事件总线
```

### 核心组件

#### 1. 目标系统 (Objectives)
- `TaskObjective` - 目标基类
- `CountObjective` - 计数型目标(击杀、收集等)
- `StateObjective` - 状态型目标(到达、装备等)

#### 2. 条件系统 (Conditions)
- `TaskCondition` - 条件基类
- `LevelCondition` - 等级条件
- 可扩展更多条件类型

#### 3. 奖励系统 (Rewards)
- `TaskReward` - 奖励基类
- `ExperienceReward` - 经验奖励
- `ItemReward` - 物品奖励

#### 4. 状态管理
- `TaskState` - 任务状态枚举
  - LOCKED (锁定)
  - AVAILABLE (可接取)
  - ACTIVE (进行中)
  - COMPLETED (已完成)
  - CLAIMED (已领奖)
  - FAILED (失败)
  - EXPIRED (过期)
  - ABANDONED (放弃)

## 快速开始

### 1. 添加到项目

将 `scripts/quest/` 目录复制到你的项目中。

### 2. 设置自动加载

在项目设置中添加自动加载:

```
TaskManager -> res://scripts/quest/task_manager.gd
QuestEventBus -> res://scripts/quest/quest_event_bus.gd
```

### 3. 创建第一个任务

```gdscript
# 在你的游戏初始化代码中
func _ready():
    # 创建任务数据
    var task = TaskData.new()
    task.task_id = "first_quest"
    task.task_name = "第一个任务"
    task.description = "击杀5只史莱姆"
    
    # 添加目标
    var objective = CountObjective.new()
    objective.objective_id = "kill_slimes"
    objective.description = "击杀史莱姆"
    objective.target_type = "kill_enemy"
    objective.target_id = "slime"
    objective.required_count = 5
    task.objectives.append(objective)
    
    # 添加奖励
    var reward = ExperienceReward.new()
    reward.experience = 100
    task.rewards.append(reward)
    
    # 注册任务
    TaskManager.register_task(task)
    
    # 设置玩家引用
    TaskManager.set_player($Player)
```

### 4. 接取任务

```gdscript
# 玩家接取任务
TaskManager.accept_task("first_quest")
```

### 5. 触发游戏事件

```gdscript
# 当击杀敌人时
QuestEventBus.emit_enemy_killed("slime", "slime_001", 1)
```

### 6. 领取奖励

```gdscript
# 任务完成后领取奖励
TaskManager.claim_rewards("first_quest")
```

## 详细使用指南

### 创建任务

#### 基础任务配置

```gdscript
var task = TaskData.new()
task.task_id = "unique_id"           # 唯一ID
task.task_name = "任务名称"           # 显示名称
task.description = "任务描述"         # 详细描述
task.category = "主线"                # 任务类别
task.priority = 10                   # 优先级(数值越大越高)
```

#### 添加目标

**计数型目标:**
```gdscript
var obj = CountObjective.new()
obj.objective_id = "obj_1"
obj.description = "击杀史莱姆"
obj.target_type = "kill_enemy"       # 事件类型
obj.target_id = "slime"              # 目标ID
obj.required_count = 10              # 需要数量
obj.optional = false                 # 是否可选
obj.weight = 1.0                     # 权重(用于计算总进度)
task.objectives.append(obj)
```

**状态型目标:**
```gdscript
var obj = StateObjective.new()
obj.objective_id = "obj_2"
obj.description = "到达城镇"
obj.state_type = "location"          # 状态类型
obj.target_state = "town"            # 目标状态
task.objectives.append(obj)
```

#### 添加条件

**等级条件:**
```gdscript
var condition = LevelCondition.new()
condition.required_level = 5
condition.compare_type = LevelCondition.CompareType.GREATER_OR_EQUAL
task.accept_conditions.append(condition)
```

#### 添加奖励

**经验奖励:**
```gdscript
var reward = ExperienceReward.new()
reward.experience = 100
task.rewards.append(reward)
```

**物品奖励:**
```gdscript
var reward = ItemReward.new()
reward.item_id = "gold_coin"
reward.quantity = 50
task.rewards.append(reward)
```

**可选奖励(玩家选择其一):**
```gdscript
var reward1 = ItemReward.new()
reward1.item_id = "sword"
task.optional_rewards.append(reward1)

var reward2 = ItemReward.new()
reward2.item_id = "shield"
task.optional_rewards.append(reward2)
```

#### 高级配置

**时间限制:**
```gdscript
task.time_limit = 600.0  # 10分钟(秒)
```

**可重复任务:**
```gdscript
task.repeatable = true
task.cooldown = 86400.0      # 24小时冷却
task.max_completions = 0     # 0表示无限次
```

**前置任务:**
```gdscript
task.prerequisite_tasks = ["task_1", "task_2"]
```

**互斥任务:**
```gdscript
task.exclusive_tasks = ["task_b"]  # 接取此任务后,task_b不可接取
```

### 使用任务管理器

#### 注册任务

```gdscript
# 单个注册
TaskManager.register_task(task_data)

# 批量注册
var tasks: Array[TaskData] = [task1, task2, task3]
TaskManager.register_tasks(tasks)
```

#### 任务操作

```gdscript
# 检查是否可接取
if TaskManager.can_accept_task("task_id"):
    # 接取任务
    TaskManager.accept_task("task_id")

# 放弃任务
TaskManager.abandon_task("task_id")

# 失败任务
TaskManager.fail_task("task_id", "超时")

# 领取奖励
TaskManager.claim_rewards("task_id")

# 领取可选奖励(传入索引)
TaskManager.claim_rewards("task_id", 0)  # 选择第一个可选奖励
```

#### 查询任务

```gdscript
# 获取任务实例
var instance = TaskManager.get_task_instance("task_id")

# 获取所有活跃任务
var active_tasks = TaskManager.get_active_tasks()

# 获取所有可接取任务
var available_tasks = TaskManager.get_available_tasks()

# 检查任务是否完成
if TaskManager.is_task_completed("task_id"):
    print("任务已完成")
```

#### 监听任务事件

```gdscript
func _ready():
    TaskManager.task_accepted.connect(_on_task_accepted)
    TaskManager.task_updated.connect(_on_task_updated)
    TaskManager.task_completed.connect(_on_task_completed)
    TaskManager.task_claimed.connect(_on_task_claimed)
    TaskManager.task_failed.connect(_on_task_failed)

func _on_task_accepted(task_id: String):
    print("任务已接取: ", task_id)

func _on_task_updated(task_id: String, progress: float):
    print("任务进度: %s - %.1f%%" % [task_id, progress * 100])

func _on_task_completed(task_id: String):
    print("任务已完成: ", task_id)
```

### 使用事件总线

#### 触发游戏事件

```gdscript
# 击杀敌人

# 收集物品

# 到达位置
QuestEventBus.emit_location_reached("town", Vector2(100, 200))

# NPC对话
QuestEventBus.emit_npc_talked("merchant", "dialogue_1")

# 自定义事件
QuestEventBus.emit_custom("custom_event", {
    "target_id": "something",
    "count": 1
})
```

#### 直接连接信号

```gdscript
func _ready():
    QuestEventBus.instance.enemy_killed.connect(_on_enemy_killed)

func _on_enemy_killed(enemy_type: String, enemy_id: String, level: int):
    print("击杀了: ", enemy_type)
```

### 保存和加载

```gdscript

# 加载任务数据
# 从文件加载 save_data...
```

## 扩展开发

### 创建自定义目标类型

```gdscript
class_name CustomObjective
extends TaskObjective

@export var custom_param: String = ""

func update_progress(event_data: Dictionary) -> void:
    # 实现自定义逻辑
    if event_data.get("type") == "custom_event":
        set_completed(true)

func check_completion() -> bool:
    return is_completed
```

### 创建自定义条件类型

```gdscript
class_name CustomCondition
extends TaskCondition

@export var required_value: int = 0

func check(context: Dictionary) -> bool:
    var player = context.get("player")
    if not player:
        return false
    
    # 实现自定义检查逻辑
    var value = player.get_custom_value()
    var result = value >= required_value
    
    return result if not negate else not result
```

### 创建自定义奖励类型

```gdscript
class_name CustomReward
extends TaskReward

@export var reward_type: String = ""

func grant(context: Dictionary) -> bool:
    var player = context.get("player")
    if not player:
        return false
    
    # 实现自定义奖励发放逻辑
    player.grant_custom_reward(reward_type, amount)
    return true

func get_preview_text() -> String:
    return "自定义奖励: %s x%d" % [reward_type, amount]
```

## 最佳实践

### 1. 任务ID命名规范

```gdscript
# 推荐格式: 类型_章节_序号
"main_01_001"      # 主线第1章第1个任务
"side_village_01"  # 村庄支线任务1
"daily_combat"     # 每日战斗任务
"achieve_killer"   # 击杀成就
```

### 2. 使用资源文件

将任务数据保存为 `.tres` 资源文件,便于在编辑器中可视化编辑:

```gdscript
# 创建资源
var task = TaskData.new()
# ... 配置任务 ...
ResourceSaver.save(task, "res://data/quests/task_001.tres")

# 加载资源
var task = load("res://data/quests/task_001.tres")
TaskManager.register_task(task)
```

### 3. 集中管理任务配置

创建任务配置管理器:

```gdscript
# quest_config.gd
class_name QuestConfig
extends Node

const QUEST_DIR = "res://data/quests/"

static func load_all_quests() -> Array[TaskData]:
    var quests: Array[TaskData] = []
    var dir = DirAccess.open(QUEST_DIR)
    
    if dir:
        dir.list_dir_begin()
        var file_name = dir.get_next()
        
        while file_name != "":
            if file_name.ends_with(".tres"):
                var quest = load(QUEST_DIR + file_name)
                if quest is TaskData:
                    quests.append(quest)
            file_name = dir.get_next()
    
    return quests
```

### 4. 任务进度UI更新

```gdscript
# quest_ui.gd
extends Control

func _ready():
    TaskManager.task_updated.connect(_update_quest_display)

func _update_quest_display(task_id: String, progress: float):
    var instance = TaskManager.get_task_instance(task_id)
    if not instance:
        return
    
    # 更新UI显示
    $ProgressBar.value = progress * 100
    
    # 更新目标列表
    for obj in instance.objectives:
        update_objective_ui(obj)
```

### 5. 性能优化

```gdscript
# 避免每帧检查任务状态
# 使用信号驱动更新

# 不推荐:
func _process(delta):
    check_all_tasks()  # ❌

# 推荐:
func _ready():
    TaskManager.task_updated.connect(_on_task_updated)  # ✅
```

### 6. 错误处理


## 常见问题

### Q: 如何创建任务链?

A: 使用 `prerequisite_tasks` 属性:

```gdscript
task2.prerequisite_tasks = ["task1"]
task3.prerequisite_tasks = ["task2"]
```

### Q: 如何实现分支任务?

A: 使用 `exclusive_tasks` 属性:

```gdscript
task_a.exclusive_tasks = ["task_b"]
task_b.exclusive_tasks = ["task_a"]
# 接取task_a后,task_b将被锁定
```

### Q: 如何追踪多个目标?

A: 在一个任务中添加多个目标:

```gdscript
task.objectives.append(objective1)
task.objectives.append(objective2)
task.objectives.append(objective3)
```

### Q: 如何实现隐藏任务?

A: 不在UI中显示,但仍然注册到系统:

```gdscript
task.tags.append("hidden")
# 在UI中过滤掉带有"hidden"标签的任务
```
