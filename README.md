# 2D ARPG 游戏项目

这是一个基于 Godot 4.x 引擎开发的 2D 动作角色扮演游戏项目，包含完整的战斗、装备、背包、任务等系统。

## 功能特性

- **战斗系统**：支持多种伤害类型、暴击、状态效果、元素反应
- **装备系统**：武器、装备槽位、属性加成
- **背包系统**：物品管理、拖拽操作、快捷栏
- **任务系统**：任务追踪、奖励发放
- **属性系统**：角色属性、自动恢复、装备加成
- **掉落系统**：物品掉落、拾取机制

## 快速开始

### 环境要求

- Godot 4.x
- Windows/Linux/macOS

### 运行项目

1. 克隆或下载项目到本地
2. 使用 Godot 编辑器打开 `project.godot` 文件
3. 点击运行按钮启动游戏

## 核心系统说明

### 战斗系统
- 支持物理、魔法、元素伤害
- 暴击、格挡、闪避机制
- 状态效果和元素反应

### 装备系统
- 武器和装备槽位管理
- 属性加成计算
- 装备更换和卸下

### 背包系统
- 物品存储和管理
- 拖拽操作界面
- 快捷栏功能

### 任务系统
- 任务创建和追踪
- 奖励发放机制
- 任务状态管理

### 属性系统
- 角色属性管理
- 自动恢复机制
- 装备和状态效果影响

## 开发指南

### 代码规范
- 使用 GDScript 编写脚本
- 遵循 Godot 命名约定
- 添加必要的中文注释
- 保持代码结构清晰

### 测试
项目包含完整的测试套件，位于 `tests/` 目录下。
